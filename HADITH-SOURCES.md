# Hadith Source of Truth

## Primary Source
**https://hadeethenc.com/ar/home**

This is the authoritative source for all hadith verification in this project. Before adding or modifying any hadith, always verify:

1. **Arabic text** — exact wording from this source
2. **Authenticity grade** — Sahih, Hasan, Da'if, etc.
3. **Narrator** — who reported the hadith
4. **Reference numbers** — Bukhari, Muslim, Abu Dawud, Tirmidhi, etc.
5. **Explanation (Sharh)** — the scholarly explanation of meaning and application

## Verification Rules
- Every hadith MUST cite its full reference (e.g., Sahih al-Bukhari 2856, Sahih Muslim 30)
- "Muttafaqun Alayh" (Agreed Upon) = both Bukhari AND Muslim narrated it
- Never paraphrase the Arabic text — use the exact words from the source
- The explanation must match the scholarly commentary, not personal opinion
